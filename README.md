# Игра "Крестики-нолики"
Это простая игра в крестики-нолики, где победитель будет тот, чьи фигуры встанут втроем в ряд или по диагонали.
## Технологии
- **React**
- **JS**
- **CSS**
- **HTML**
## Инструкция
1. Склонируйте репозиторий
```shell
git clone https://github.com/powder-rose/Tic-Tac-Toe.git
```
2. Установите зависимости
```shell
npm i
```
3. Готово! Можете запустить с помощью дев сервера.
##

Запустить дев сервер:

```shell
npm run start
```
Сборка:

```shell
npm run build
```
